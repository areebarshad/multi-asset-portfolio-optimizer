# Plots Directory

This directory contains plots generated from the Multi-Asset Portfolio Optimizer model.

## Folder Breakdown

| Folder | Description |
| ------ | ----------- |
| `Closing_Prices/` | Line plots showing daily closing prices for all 15 ETFs over 2015–2025. |
| `Daily_Return_Distributions/` | Histograms of daily log returns across all assets. |
| `Returns_Correlation_Heatmap/` | Heatmap of pairwise correlations between asset daily returns. |
| `Optimal_Weights/` | Bar chart of tangency portfolio (max Sharpe) weights per asset. |
| `Efficient_Frontier/` | Efficient frontier with gross curve, net-of-10bps-cost curve, Capital Market Line, and tangency point. |
| `Cumulative_Returns/` | Cumulative return comparisons: optimized portfolio, equal-weight portfolio, and SPY benchmark. |
| `Walk_Forward_Backtest/` | Walk-forward backtest cumulative returns vs equal-weight and SPY (BL+LW weights re-estimated per period). |

## Notes

- All plots are generated by running `notebooks/Final_Model/model.py`.
- The final model uses a 15-asset ETF universe, a tangency (max Sharpe) portfolio formulation, Ledoit-Wolf shrinkage, and a Black-Litterman prior — plots reflect this expanded methodology vs the original 8-asset min-variance model.
